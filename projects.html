Pipelined Processor Simulatorclose

A pipelined processor simulator that runs LC-2200 machine code instructions. LC-2200 is a MIPS-like language used in CS2200 at Georgia Tech.

The 5 stages are:

  1. Instruction Fetch
  2. Instruction Decode/Register Read
  3. Execute
  4. Memory Access/Write
  5. Write Back

Made in Logisim.

Malloc/Free Algorithm Simulatorclose

A simulator of C's malloc and free algorithms for efficiently allocating and freeing memory. Given a chunk of memory and a request for n bytes, the simulator allocates pieces of it dynamically by maintaining an internal free list. Two internal representations of the free-list are simulated; address ordering and size ordering.

Made in C.
